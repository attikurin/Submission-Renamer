提出物リネーム屋さん 📝

「誰のファイルか分からない...」を解決する、教育現場向けのブラウザ完結型ファイル整理ツールです。
タブレット等で回収したバラバラなファイル名の画像やドキュメントを、名簿リストやルールに従って一括でリネームし、整理されたZIPファイルとして保存できます。

デモ: こちらからアプリを開く （※URLは例示です。実際のデプロイ先に変更してください）

✨ 特徴

完全ローカル処理で安全・安心 🛡️

サーバーへのアップロードは一切行いません。すべての処理はお使いのブラウザ（クライアントサイド）内で完結します。

児童・生徒の個人情報や成績に関わるデータも安心して取り扱えます。

3つのリネームモード

名簿リストモード: Excel等の名簿（出席番号＋氏名など）をコピペするだけで、ファイルの並び順通りに名前を割り当てます。

連番・ルールモード: 「提出物_001」のように、接頭辞・接尾辞・連番桁数を指定して整理できます。

置換モード: ファイル名に含まれる特定の文字を一括で置き換えます。

直感的な操作

ドラッグ＆ドロップでファイルを追加。

リスト内でドラッグして順番を入れ替え可能。

インストール不要

HTMLファイル1つで動作するため、学校のPCやタブレットですぐに使えます。

🚀 使い方

ファイルを選択

整理したいファイルをドラッグ＆ドロップ、またはクリックして選択します。

リネーム設定

名簿リスト: Excelなどから「出席番号 名前」の列をコピーして貼り付けます。上から順に適用されます。

連番・ルール: 共通のファイル名を付けたい場合に設定します。

置換: 不要な文字列を削除したり変更したりする場合に使います。

設定後、「プレビューに反映」を押すと右側のリストが更新されます。

保存

「ZIPでまとめて保存」ボタンを押すと、リネームされたファイルが入ったZIPファイルがダウンロードされます。

⚠️ トラブルシューティング

Q. ダウンロード時に「安全にダウンロードできません」等の警告が出る
A. ブラウザ上で動的にZIPファイルを生成しているため、Windowsやブラウザのセキュリティ機能（SmartScreen等）が反応することがあります。ウイルス等は含まれていません。
警告が出た場合は、**「保存」または「継続」**を選択してください。

Q. ZIPファイルを解凍（展開）できない
A. Windowsのセキュリティ機能によりブロックされている可能性があります。

ダウンロードしたZIPファイルを右クリックし「プロパティ」を選択。

「全般」タブ下部の「セキュリティ」項目にある「許可する (Unblock)」にチェックを入れてOKを押す。

その後、通常通り展開してください。
